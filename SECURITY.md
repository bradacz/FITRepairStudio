# Security Policy

## Supported Versions

Security fixes target the current `main` branch and the latest notarized macOS
release.

## Reporting A Vulnerability

Please do not open public GitHub issues for vulnerabilities that could expose
private files, local paths, credentials, or payment configuration.

Report security issues privately:

```text
info@mylocalio.com
```

Include:

- A short description of the issue.
- Steps to reproduce.
- Affected version or commit.
- Whether private FIT data, local files, or credentials could be exposed.

## Private Activity Data

FIT files may contain sensitive GPS, health, and device data. Do not send real
activity files unless they are required to reproduce the issue and you are
comfortable sharing them privately with Localio Labs s.r.o.
